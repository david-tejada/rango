Rango was installed!

Rango is a cross browser extension that helps you interact with your browser using your voice and Talon. It can also be used as a standalone extension to click elements using your keyboard. It works by drawing hints with letters next to elements that you can use to click, hover, copy or show link addresses. It also helps you scroll, open multiple links in new tabs, close multiple tabs and more.

Use With Keyboard

In order to use Rango with your keyboard you need to first activate Keyboard Clicking. You can do this by pressing the designated shortcut ctrl-shift-5 or by right clicking on the extension button on the toolbar and then clicking on the Keyboard Clicking menu.

Once activated, to click an element you just need to type the corresponding hint letters.

Use With Talon

In order to use Rango with Talon you need to add rango-talon to your Talon user folder. Clone or download the rango-talon files to your Talon user folder.

Basic Usage

To click on elements you just have to say the word click followed by the letters of the hint. You can also omit the word click using Direct Clicking. By default Direct Clicking is active. You can deactivate it with the command rango explicit. With Explicit Clicking you have to precede every hint with the word click. This mode prevents any misclicks at the expense of being a bit more verbose.
